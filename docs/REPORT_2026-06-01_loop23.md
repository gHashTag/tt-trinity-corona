## Research: t27→Silicon Pipeline Uniqueness

Literature review confirms no prior work combining:
1. **Spec-first numeric format metadata ROM** on silicon (80 formats, 10 bytes each)
2. **On-die format converters** (18 RTL decoders covering FP16/BF16/TF32/FP8/FP4/INT4/BitNet/BCD/Posit/E8M0/MXFP/MXINT)
3. **Deterministic code generation** from `.t27` specs to Verilog (multi-language parallel corpus)

Closest related work:
- **Google TPU**: Custom numeric formats (BF16) but no format oracle
- **IBM POWER**: Hardware decimal FP (BCD) but single-format, not multi-format
- **Posit arithmetic units** (Gustafson et al.): Single-format, not catalog-based
- **NVIDIA Transformer Engine**: FP8 E4M3/E5M2 support, but format-specific, not universal

Corona is unique as a **conformance oracle** — it doesn't compute, it
characterizes. The ROM provides machine-readable format metadata (bit widths,
bias, flags) while decoders prove correctness by converting to IEEE FP32.
