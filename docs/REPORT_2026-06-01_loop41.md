### 2. Research Findings

**YosysHQ EQY tool:** Dedicated RTL-vs-netlist formal equivalence checker. For combinational decoders, this can replace GLS entirely — proving post-synthesis correctness without simulation. Viable next-level verification for the oracle.

**MXDOTP / Vmxdotp (2025-2026):** RISC-V ISA extensions for OCP MX dot products. Defines FP9 (E5M3) as intermediate accumulation format — potential future catalog addition.

**SDF-annotated GLS impractical on open-source PDKs:** Missing specify blocks in GF180MCU cells. For combinational oracle, zero-delay GLS + formal equivalence is sufficient.
