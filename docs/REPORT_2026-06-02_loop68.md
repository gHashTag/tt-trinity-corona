## Scientific Research

### FP4 Production Validation (3 New Papers)

- **Attn-QAT** (UCSD, Mar 2026, arXiv:2603.00040): First systematic FP4 QAT for attention. E2M1 format. Naive drop-in is unstable; fused Triton kernels fix it. 1.5x speedup on RTX 5090. Directly validates Corona's FP4 decoder.
- **NVIDIA QAD** (Jan 2026, arXiv:2601.20088): Quantization-Aware Distillation for NVFP4 (E2M1 + E4M3 block scales). 99.4% of BF16 accuracy. Corona's fp4 + e8m0 + mxfp4 ROM entries cover this format stack.
- **FP4 All the Way** (Intel Gaudi, May 2025, arXiv:2505.19115): First fully FP4-quantized training at 200B tokens. 7B model on 256 Gaudi2 accelerators matches BF16. Uses NVFP4 = E2M1 format family.

### Production Silicon Validation

- **AMD MI355X** (ISSCC 2026, Session 2): Ships native FP4, FP6, FP8, FP16, FP64. 20.1 PFLOPS at FP4/FP6. Selective sharing architecture. **Validates 5 Corona-cataloged formats in production silicon.**
- **DHFP-PE** (IIT Indore, Apr 2026, arXiv:2604.04507): Dual-precision MAC for FP8 (E4M3/E5M2) + FP4 (E2M1/E1M2). 28nm, 1.94 GHz. Demonstrates the FP4 E1M2 variant alongside E2M1.
- **FlexiBit** (UC Irvine, Nov 2024, arXiv:2411.18065): Bit-parallel accelerator for arbitrary FP/INT precision. 1.66x perf/area on GPT-3 in FP6. Validates demand for format diversity.
