### 3. Research Findings

**FSM DONE-state timing**: The 2-cycle STATUS→DONE→IDLE transition is inherent to the FSM design — DONE is a separate state that clears output registers before returning to IDLE. This is standard practice to prevent output glitches between transactions.

**GLS as synthesis canary**: Gate-level simulation catches technology mapping bugs that RTL simulation and formal verification miss. The FP4 and posit8 tests exercise the full CMD/DATA/STATUS protocol path through synthesized gates, verifying that Yosys `synth -flatten` preserves functional correctness for these decoders.

**Posit8 GLS value**: The posit8 decoder is the most complex Tier-1 converter (256-entry LUT, regime/exponent/fraction extraction). Its GLS test validates that the synthesis tool correctly maps the case statement through ABC optimization.
