## Implementation

### Bug Fix: `_float_to_u32` NameError

The `_ref_mxint8` function had an `except ImportError` fallback referencing `_float_to_u32()` which was never defined. Although dead code (MicroPython has `struct`), this was a latent defect. Fixed by removing the try/except wrapper entirely.

### BF16 Full Exhaustive Sweep

Added `_sweep_bf16()`: iterates all 65,536 two-byte inputs, verifies each against `_ref_bf16(val) = val << 16`. On RP2350, estimated ~65 seconds.

### TF32 Boundary + Random Sweep

Added `_sweep_tf32()`: tests boundary values (zero crossings, sign flip, Inf/NaN edges) plus 1,024 random samples from the 19-bit space. Total ~1,280 unique values. Full 524,288-value sweep would take ~9 minutes — the boundary+random approach catches any systematic decode error.
