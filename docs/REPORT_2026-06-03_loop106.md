## Research (this loop, web available)

A documentation landing page / "diataxis"-style index is a recognised remedy for
a docs tree that has accreted artifacts faster than navigation: readers need a
map that distinguishes reference material (specs, ADRs, evidence) from a
chronological log. The index applies the same link-don't-duplicate discipline used
across this arc -- it points to the authoritative files rather than restating
them, so it stays correct as long as those files exist (every link was verified).

Sources: [Diataxis documentation framework](https://diataxis.fr/),
[README / docs landing-page conventions](https://www.writethedocs.org/guide/).
