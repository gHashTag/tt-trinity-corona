## Research Findings

### Formally Verified Open-Source Tapeouts
No formally verified Tiny Tapeout submission found in literature or GitHub.
A CRC core and FIR filter have combined SymbiYosys proofs with SKY130 GDS,
but no project achieves Corona's scope (19 proofs + k-induction + 39 cocotb).
**Corona would be the first formally verified TT submission.**

### GF180MCU Timing
Comparable RISC-V designs on IHP 130nm achieve 72 MHz. GF180MCU at 180nm/5V
should support tens of MHz. Corona's 50 MHz target is feasible — the design
is mostly combinational decoders with a simple 5-state FSM.

### ROM Implementation
OpenRAM does not support GF180MCU. Case-statement ROM is optimal for <256
entries (Corona has 80). Standard-cell synthesis is the only practical option
and is area-efficient at this scale.

### Format Conversion Proofs
No new work beyond FLoPS/Wintersteiger. POSAR (posit arithmetic for RISC-V)
provides accuracy data but no formal proofs. Corona's 17 formally verified
decoders remain unique in the literature.
