## Bugs Fixed

### RTL Bugs (3)

1. **Output mux anchor conflict**: `is_anchor_cmd` (combinational, always-on)
   overrode STATUS readback whenever the last data byte was `0x7F`. Fix:
   swap priority so `ST_STATUS` is checked before `is_anchor_cmd`.

2. **Sub-byte decoder bit positions**: Right-shift register puts 1-byte data
   at `[31:24]`, so sub-byte formats must read low bits of that byte:
   - FP4/NF4 (4-bit): `[31:28]` → `[27:24]` (low nibble)
   - FP6 E3M2/E2M3 (6-bit): `[31:26]` → `[29:24]` (low 6 bits)

3. **Posit8 LZC casez ordering**: The `casez` matched `7'b0??????` first,
   returning `lzc=1` for ALL inputs starting with 0. Correct order:
   most-specific (most leading zeros) first. This broke all posit8 values
   with regime run length > 1.

### Test Infrastructure Bugs (2)

4. **Reset triggers CMD1**: `reset_dut` set `ui_in=0` (bit7=0), which the
   FSM interprets as CMD1 during post-reset idle clocks. Fix: `ui_in=0x80`.

5. **Read timing off by one clock**: `read_result_bytes` advanced the clock
   before sampling, so `status_cnt` was already incremented. Fix: sample
   combinational output (Timer 1ns) before advancing (RisingEdge).
