## Research: Scientific Literature (Loop 18)

### BitNet 1.58b Ternary Encoding
No IEEE-style standard. Weights are {-1, 0, +1} in 2-bit slots. NativeTernary
(Apr 2026) achieves 2.000 bits/weight self-delimiting encoding. Hardware exists:
BitROM (Sep 2025) stores two ternary weights per nMOS transistor. TeLLMe (Apr
2025) is an FPGA pipeline at <7W. **Actionable**: Trivial 2-bit LUT decoder,
~4 cells, would cover fmt_id 71.

### LNS for ML Inference
DATE 2025 papers on "Segment-Wise Accumulation" and "Lookup Table Refactoring"
address LNS addition cost. 4-5 bit LNS inference shows negligible degradation.
**Actionable**: Our LNS8 decoder (88 cells) is already ahead of the curve.

### Formal Verification with SymbiYosys
SymbiYosys (SBY) uses Yosys + SMT solvers for bounded model checking. For our
8-bit decoders (256 input states, combinational), BMC can exhaustively prove
correctness in seconds. **Actionable**: Write SVA `assert` properties in each
decoder module; run `sby` to formally verify all 256 patterns match reference.

### Post-Silicon Validation
Spacely (Fermilab, arXiv 2406.15181) is an open-source framework for small ASIC
teams, supporting NI-PXI and Caribou platforms. TT ecosystem spans 3 fabs with
1000+ designs shipped in 2025. **Actionable**: Plan post-fab validation around
TT carrier board + our cocotb test suite adapted for physical SPI/GPIO.

### Mixed-Precision Training State of Art
E4M3 forward + E5M2 gradients is the dominant paradigm. MXFP8 uses per-32-element
block scaling. Our oracle covers both element types. No new element format needed.
