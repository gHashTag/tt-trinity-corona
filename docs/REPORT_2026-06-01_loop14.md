## Implemented

### 1. E8M0 Decoder (`src/rtl/e8m0_decode.v`)
- 8-bit exponent-only format, no sign/mantissa
- OCP MX v1.0 shared block scale factor
- Value = 2^(e - 127), bias 127, same as FP32 exponent field
- Special: 0xFF = NaN (entire MX block becomes NaN), 0x00 = 2^(-127)
- FP32 conversion: `{0, e, 23'b0}` for normal, subnormal for e=0
- 256/256 exhaustive test pass

### 2. MXINT8 Decoder (`src/rtl/mxint8_decode.v`)
- 8-bit two's complement with implicit 2^(-6) scale
- Q1.6 fixed-point: value = signed_int / 64
- Range: [-1.984375, +1.984375], -128 reserved → NaN
- Conversion via 7-bit leading-one priority encoder → FP32 exp + mantissa
- 256/256 exhaustive test pass

### 3. ROM Expansion (78 → 80 records)
- fmt_id=78: E8M0 (cluster 5, OCP MX, enc=MX, FLAG_ON_DIE)
- fmt_id=79: MXINT8 (cluster 5, OCP MX, enc=MX, FLAG_ON_DIE)
- ROM readback: 80/80 records verified
