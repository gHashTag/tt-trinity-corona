## Research Summary (Loop 19)

### BitNet Hardware — Production Path Emerging
- **BitROM** (Sep 2025): 2 ternary weights per nMOS transistor in bidirectional ROM
- **TeLLMe** (Apr 2025): FPGA pipeline using table-lookup GEMM at <7W
- **NativeTernary** (Apr 2026): 2.000 bits/weight self-delimiting encoding
- No IEEE-style standard yet; 2-bit packed encoding is most common

### SymbiYosys Formal Verification
Open-source BMC (Yosys + SMT solvers) can prove 8-bit decoder correctness
exhaustively in seconds. Write SVA `assert` properties inline. **Not available
locally** (sby not installed), but CI can run it in a Docker container.

### Spacely Post-Silicon Framework
Fermilab's open-source framework for small ASIC teams. Supports NI-PXI and
Caribou test platforms. Directly usable for post-fab validation of Corona.
