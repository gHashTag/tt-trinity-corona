## Research (this loop, web available)

Hand-typed counts in prose are the classic drift surface: each is a copy of a
number whose source lives elsewhere, and copies rot (here PLAN and README
disagreed on formal tasks; the README carried a stale GL and decoder count). The
durable fix is to derive (as the verification dossier now does for gates/codes,
Loop 115) or, where prose must state a number, to audit it against the artifact
and keep one canonical figure. This sweep closes the last hand-maintained counts;
the dynamic ones (gates, codes, decoders) are already generated and freshness-gated.

Sources: [single source of truth / docs drift](https://dev.to/codinsonn/the-joy-of-single-sources-of-truth-277o),
[documentation testing (counts vs artifact)](https://www.writethedocs.org/guide/).
