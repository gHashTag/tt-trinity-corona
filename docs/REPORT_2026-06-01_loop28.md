## Research Findings

### IEEE P3109
Still unratified draft. FLoPS (Lean 4, arXiv 2602.15965 v3) and Wintersteiger
(ARITH 2025) formally verify P3109 decode — same property class Corona proves
via SymbiYosys/z3.

### OCP Microscaling
MX v1.0 defines MXFP8/MXFP6/MXFP4/MXINT8 with E8M0 scales. NVIDIA NVFP4
diverges (16-element blocks, two-level scaling). M2XFP (ASPLOS 2026) adds
metadata augmentation. Corona already covers MXFP8, MXINT8, E8M0, FP6, FP4.

### Low-Bit Quantization
New formats: any4 (learned codebook), NVINT4, Apple A19 MXFP4 hardware,
Google TurboQuant. ByteDance shows MXINT8 can outperform MXFP8 with rotation.

### Open-Source Silicon
LibreLane (community OpenLane 2 rewrite, July 2025) supports GF180MCU.
wafer.space runs GF180MCU shuttles. Tiny Tapeout sent 1000+ designs to 3 fabs
in 2025.
