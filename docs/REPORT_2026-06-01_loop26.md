1(CMD2) + 15(DATA max) + 10(ROM STATUS) + 1(DONE) = 27 cycles.

## Research: Key Findings

### SymbiYosys FSM Verification Best Practices
- BMC depth = max cycle path + margin (28+2=30 for Corona)
- `initial assume(!rst_n)` required for async-reset designs
- k-induction (`mode prove`) needs auxiliary invariants for completeness
- ZipCPU tutorials confirm this methodology is production-standard

### IEEE P3109 Status
Not yet ratified (still working group draft). P3109 defines parametric ML FP
formats (<16 bits). Corona's FP4/FP6/FP8/BF16 decoders are P3109-adjacent
but follow OCP/vendor specs. FLoPS (Lean 4) and Wintersteiger (ARITH 2025)
formally verify the P3109 `decode` function — same property Corona proves.
