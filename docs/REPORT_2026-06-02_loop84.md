## Research (this loop, web available)

The Lucas/golden-ratio identity is rigorously established: `L_n = phi^n + (-phi)^-n` (Binet), so for even n, `phi^n + phi^-n = L_n`; n=2 gives `phi^2 + phi^-2 = 3 = L_2`, **exact, not an approximation**. Algebraic proof: `phi^2 = 1+phi`, `phi^-2 = 2-phi`, sum `= 3`.

Sources: [Lucas number (Wikipedia)](https://en.wikipedia.org/wiki/Lucas_number), [Fibonacci/Lucas/phi formulae (Knott)](https://r-knott.surrey.ac.uk/fibonacci/FibPhiLucasFormulae.PDF), [Fibonacci and Lucas Identities the Golden Way (arXiv:1810.12115)](https://arxiv.org/pdf/1810.12115).
