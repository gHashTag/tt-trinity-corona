## Research: Numeric Format Updates

| Format | Update |
|--------|--------|
| NVFP4 | NVIDIA diverged from OCP MX: E2M1 + E4M3 scale per 16 elements (vs MX's E8M0 per 32) |
| HiFloat4 | New 64-element hierarchical format, outperforms NVFP4 on accuracy |
| eXmY (Google) | Arbitrary bit-width format with perfect compression for 3/5/6/7/9-bit |
| E8M0 | MX block scale factor format — standalone decoder candidate |
| IEEE P3109 | Still not ratified (Interim Report v3.0, Aug 2025) |
