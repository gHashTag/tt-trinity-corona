## Research (this loop, web available)

Discoverability is the other half of a documentation index: a landing page only
helps if the front page links to it. Standard README conventions place a short
"documentation" pointer near the top of the project README so a reader reaches the
map in one hop. This is the link-don't-duplicate discipline applied once more --
the root README points to `docs/README.md` rather than restating it.

Sources: [Write the Docs -- README/landing conventions](https://www.writethedocs.org/guide/),
[Diataxis documentation framework](https://diataxis.fr/).
