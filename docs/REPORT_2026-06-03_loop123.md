## Research (this loop, web available)

The lesson is to distrust a documented "can't be done" rationale and re-derive it:
the "open conjecture" tag had frozen GF256 out of verification for the whole arc,
yet a one-line bias computation showed the convention held and the actual fault was
a mundane off-by-one in the special-exponent sentinel. This is the classic
all-ones-exponent reservation (IEEE-754): the largest field value must be the
Inf/NaN code, so a sentinel set one past the field maximum is unreachable and
silently disables special handling -- detectable only by a round-trip / specials
test, never by a finite-value probe alone. Sources: IEEE-754 (reserved exponent
encodings); the same exact-rational-reference discipline as prior loops (gf256
exponents ~2^96 mandate exponent-comparison bounds, never materialized powers).
